## Temática
El proyecto consiste en el desarrollo de una API REST para descubrir y consultar instalaciones deportivas, cuyo propósito es facilitar a los usuarios encontrar lugares donde practicar distintos deportes según el tipo de instalación, el deporte disponible o la ubicación geográfica. La API utiliza un dataset de más de 15.000 instalaciones deportivas obtenido de datos abiertos de OpenStreetMap, que incluye información como nombre, tipo de instalación, deportes disponibles y coordenadas geográficas. Estos datos se almacenan en MongoDB y se exponen mediante endpoints que permiten realizar búsquedas, filtrado por deporte o tipo de instalación y paginación de resultados, con el objetivo de ofrecer una herramienta que ayude a las personas a descubrir nuevas instalaciones deportivas cercanas y fomentar la práctica de actividad física de forma accesible y sencilla.
